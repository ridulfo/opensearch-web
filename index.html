Open Search

Democratize the index

A personal, non-commercial research project exploring independent search infrastructure in Sweden

Introduction

Europe has only one truly independent search engine with its own index: Mojeek, based in the UK. The European Search Perspective (EUSP) (France & Germany) will provide a second option once released.

This limited diversity creates significant risks. When a handful of companies control how billions of people access information, we may face issues of censorship, algorithmic bias, and surveillance capitalism. Search engines shape public discourse and knowledge discovery. This power shouldn't be concentrated in so few actors.

This project aims to investigate the challenges faced when building a search engine hosted entirely in Sweden using local and/or Swedish cloud providers. While this presents challenges, as Swedish providers currently lag behind international counterparts in both usability and affordability, we believe exploring this path will drive learning and improvement for all stakeholders, ultimately contributing to a more diverse and resilient global information ecosystem.

Milestones

1. Crawl and index all HTML pages on Swedish domains (.se)
2. Build the ranking system and search engine interface
3. Expand indexing to additional top-level domains
